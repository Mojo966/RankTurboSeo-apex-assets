# RankTurboSeo Apex Assets

JavaScript modules, SVG sprites, and CSS partials for **RankTurboSeo.Apex.v16** Blogger templates.

**CDN:** https://cdn.jsdelivr.net/gh/Mojo966/RankTurboSeo-apex-assets@main/assets/
**Version:** v16.0.0
**License:** MIT

## Quick Start

```html
<script defer src="https://cdn.jsdelivr.net/gh/Mojo966/RankTurboSeo-apex-assets@main/assets/core.js"></script>
<script defer src="https://cdn.jsdelivr.net/gh/Mojo966/RankTurboSeo-apex-assets@main/assets/index.js"></script>
```

The auto-loader picks up which modules are needed based on data attributes in your DOM.
